About Me

This is an assignment for NextAcademy Front End Web Development class for lesson css layout assignment option 1.

The assignment fulfills the following conditions:

  1. Has a navigation bar
  2. Has a side bar
  3. Has a footer
  4. Has a grid of images / elements
  5. Mobile/Web view compatible with transition at 768px width so that it's view friendly at ipad size and above

The header background, nav bar buttons, nav bar text and profile photo animation were done with well timed CSS animation background-color changes & border-shadow colors.

The sidebar profile photo is always in a perfect circle as the image itself and border-radius is describe in % of its parent element.

The side bar elements, contact me bubbles and gallery were done using the flex box method.

The dissapearing sidebar during mobile view and appearing of content in the about me section during mobile view - which dissapear/reappear during web view vise versa was done using display: hide and otherwise.
